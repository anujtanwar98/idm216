Micro-interactions

View Menu Button

Triggers

  • Hover On/Off Button
  • Click On/Off Button

Rules

Front-End

Hover on Button

  • Background color changes from #FFF1D9 to #FFCE71
  • Text Color changes from #463A6C to #fff
  • Cursor changes from default to pointer

Hover off Button

  • Background color changes from #FFCE71 to #FFF1D9
  • Text Color changes from #fff to #463A6C
  • Cursor changes from pointer to default

Click on Button

  • Background Color changes from #FFCE71 to #f6b338
  • Button translates 2px down on the Y-axis

Click off Button

  • Background Color changes from #f6b338 to #FFCE71
  • Button translates 2px up on the Y-axis back to original position
  • Cursor changes from progress to pointer

BackEnd

  • Once Button is clicked, system will check and load menu information from the Menu Items Table in our database

Menu Item Section

Triggers

  • On Click Hold
  • On Drag (Horizontally)
  • Click on Menu Item

Rules

Front-End

On click Hold

  • Cursor changes from Default to Grab

On Drag

  • Cursor changes from Grab to grabbing
  • Menu Items move in direction of drag (Left to right)

On-Click Release

  • Cursor changes from grabbing to default

Click on Menu Item

  • Cursor changes from default to progress
  • Modal opens with Menu Item information

BackEnd

  • When Click On Menu Item, system must retrieve Menu Item information from Database

Quantity section

Triggers

  • Hover On/Off Plus/Minus Buttons
  • Click On/Off Plus Button
  • Click On/Off Minus Button
  • Click On/Off Quantity Box

Rules

Front-End

Hover On Plus/Minus Buttons

  • Cursor changes from default to pointer
  • Background Color changes from #FFF1D9 to #463A6C
  • Color changes from #463A6C to #FFF1D9

Hover Off Plus/Minus Button

  • Cursor changes from pointer to default
  • Background Color changes from #463A6C to #FFF1D9
  • Color changes from #FFF1D9 to #463A6C

Click On/Off Plus Button

  • When clicked, button transforms with a scale of 0.95 with a #463A6C drop-shadow of 10px. When clicked off button, it goes back to original state.
  • When the button is clicked, the quantity increases

Click On/Off Minus Button

  • When clicked, button transforms with a scale of 0.95 with a #463A6C drop-shadow of 10px. When clicked off button, it goes back to original state
  • When the button is clicked, the quantity decreases

Hover On/Off Quantity Box

  • Cursor Change from Default to Pointer
  • When Hover on, border of Quantity Box will change to 2px solid #463A6C. When Hover Off, Border of Quantity Box will change back to 2px solid black

Click On Quantity Box

  • Can change quantity by typing number in
  • Will have an additional border of 1px solid #FFF1D9 color

Click Off Quantity Box

  • Cursor changes from pointer to default
  • Quantity Box's border goes back to original state and does not have additional border of 1px solid in #FFF1D9 color

BackEnd

  • When quantity is changed, the system will save the quantity the user selects and send the information to the food truck.

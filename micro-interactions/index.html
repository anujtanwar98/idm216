View Menu Button

Triggers

  • Hover On/Off Menu
  • On Click Hold Button

    • On-Click

  • On Click Release Button

Rules

Front-End

Hover on Button

  • Background color changes to (insert hex code here) with a drop-shadow of (#)px along the bottom in (Drop shadow color)
  • Cursor changes from default to pointer

Hover off Button

  • Background color changes to (insert hex code here) with no drop shadow
  • Cursor changes from pointer to default

On Click Hold Button

  • Background Color changes (slight change) with a border stroke of (insert hex code)
  • Cursor changes from pointer to progress
  • Button translates (#)px down from original position
  • Button Text Color Changes to ( New Text color)

On Click Release Button

  • Cursor changes from progress to pointer
  • Button translates (#)px up to original position
  • Button text changes to a loading spinner that is rotating
  • Loads Menu Page

BackEnd

  • Once Button is clicked, system will check and load menu information from the Menu Items Table in our database

  • On Drag (Horizontally)

Front-End

  • Cursor changes from Grab to grabbing
  • Menu Items move in direction of drag (Left to right)

On-Click Release

  • Cursor changes from grabbing to default

Click on Menu Item

  • Cursor changes from default to progress
  • Modal opens with Menu Item information

BackEnd

  • When Click On Menu Item, system must retrieve Menu Item information from Database

Quantity section

Triggers

  • Hover On/Off Plus/Minus Buttons
  • Click On/Off Plus Button
  • Click On/Off Minus Button
  • Click On/Off Quantity Box

Rules

Front-End

Hover On/Off Plus/Minus Buttons

  • Cursor changes from default to pointer

Click On/Off Plus Button

  • Button Color Change from ( HexCode) to (HexCode)
  • When clicked, button (either translates or drop shadow)
  • When clicked off button, it goes back to original state.

Click On/Off Minus Button

  • Button Color Change from ( HexCode) to (HexCode)
  • When clicked, button (either translates or drop shadow). When clicked off button, it goes back to original state.
  • When button is clicked, the quantity decreases

Hover On/Off Quantity Box

  • Cursor Change from Default to Pointer

Hover On/Off Quantity Box

  • Cursor change from Pointer to Text Quantity Box has a Border of (#)px solid (HexCode) Can change quantity by typing number in
  • Quantity Box has a Border of (#)px solid (HexCode)
  • Can change quantity by typing number in

Hover On/Off Quantity Box

  • Cursor Change from text to Pointer
  • Quantity Box's border goes back to original state

BackEnd

  • When quantity is changed, the system will save the quantity the user selects and send the information to the food truck.
